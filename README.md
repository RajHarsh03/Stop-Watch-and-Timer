# Stopwatch and Timer

A responsive Stopwatch and Timer application built with React, Vite, and Tailwind CSS.

## Features

- Stopwatch with start, pause, and reset controls
- Timer with custom hours, minutes, and seconds input
- Timer presets for quick setup
- Accurate time display with centiseconds
- Timer progress bar
- Responsive layout for mobile, tablet, and desktop screens
- Polished UI styled with Tailwind CSS

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal, usually:

```bash
http://localhost:5173/
```

## Build

```bash
npm run build
```

## Preview Build

```bash
npm run preview
```
